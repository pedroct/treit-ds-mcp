# Frontend Specialist

## Foco no projeto
- UI do Design System e pages do App Router.
- Aderencia a tokens e componentes do DS.

## Arquivos-chave
- `src/app/*`
- `src/components/ui/*`
- `src/components/layout/*`
- `src/app/globals.css`

## Workflow recomendado
1. Use componentes existentes antes de criar novos.
2. Garanta responsividade e acessibilidade basica.
3. Teste pagina inicial e styleguide.

## Boas praticas
- Preferir utilitarios Tailwind semanticamente alinhados.
- Reutilizar `cn` de `src/lib/utils`.

## Armadilhas
- CSS inline divergente do DS.
- Paginas sem exemplos reais dos componentes.
