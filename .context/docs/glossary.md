# Glossario

- **DS (Design System)**: Conjunto de componentes, tokens e padroes que padronizam UI.
- **MCP (Model Context Protocol)**: Protocolo para expor recursos/ferramentas a agentes.
- **Tokens**: Variaveis de design (cores, espacamento, tipografia).
- **Tokens semanticos**: Tokens nomeados por significado (ex.: `--primary`).
- **App Router**: Roteamento do Next.js baseado em `src/app`.
- **Provider (MCP)**: Modulo que extrai dados do repo e gera recursos.
- **Tool (MCP)**: Operacao acionavel (busca, scaffold, validacao).
- **Catalogo de componentes**: Lista indexada de `src/components`.
- **Pattern**: Exemplo de composicao de componentes em paginas do DS.
- **Health check**: Endpoint simples para validar disponibilidade (`/health`).
- **SSE**: Server-Sent Events usado no MCP para ping de conexao.
- **Shadcn UI**: Base de componentes adaptada dentro de `src/components/ui`.
